# Project Title
Pokemon App
## Description
Displays:
1. Pokemon Name
2. Pokemon Height
3. Pokemon Weight
4. Pokemon Types
5. Pokemon Image

Pokemon details like height, weight, types and image can be searched using pokemon name.
